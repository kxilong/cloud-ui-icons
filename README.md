## 安装

```bash
npm i @cloudui-design/icons
```

## 基本用法

首先，您需要全局地导入样式文件。

```js
import "@cloudui-design/icons/icomoon/cloudui-icon.css";
```

之后，您可以在 Vue 组件中简单地使用图标 :

```jsx
<i class="icon icon-alert-triangle"></i>
```

## 构建项目

```bash
npm run compile # Build library
```
